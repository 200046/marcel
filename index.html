Mes cours d'informatique

Html

HTML est le langage standard pour la création de pages web. Utilisant des balises pour définir la structure, il permet d'organiser le contenu en titres, paragraphes, images, liens, etc. Les navigateurs interprètent le code HTML pour afficher des pages web structurées.

Voir la documentation

Css

CSS est un langage de style utilisé pour embellir et mettre en page les pages web créées en HTML. En définissant des règles de style, CSS permet de contrôler la couleur, la police, la mise en page, et autres aspects de la présentation. Il assure une séparation claire entre la structure (HTML) et le style.

Voir la documentation

Python

Python, langage de programmation polyvalent, se distingue par sa syntaxe lisible et sa simplicité. Il est utilisé dans divers domaines tels que le développement web, la science des données et l'automatisation. Sa communauté active et ses bibliothèques étendues en font un choix populaire pour les développeurs.

Voir la documentation

javascript

JavaScript est un langage de programmation côté client qui permet d'ajouter des fonctionnalités interactives aux pages web. Exécuté dans le navigateur, il peut manipuler le contenu, répondre aux événements et communiquer avec le serveur. En complément d'HTML et CSS, JavaScript offre une expérience utilisateur dynamique.
